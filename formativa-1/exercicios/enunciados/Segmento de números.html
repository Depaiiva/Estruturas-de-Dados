Segmento de números

Você tem duas tarefas para este exercício.

  1. Escreva uma função
int encaixa(int a, int b)

que, recebendo dois números inteiros aa e bb, verifica se bb corresponde aos últimos dígitos de aa. Se sim, sua função deve retornar 1, se não, 0.

Exemplo 1

encaixa(567890,890)

deve retornar

1

Exemplo 2

encaixa(1243,1243)

deve retornar

1

Exemplo 3

encaixa(2457,245)

deve retornar

0

Exemplo 4

encaixa(457,2457)

deve retornar

0
  1. Usando a função encaixa do item 1, faça uma função
int segmento(int a, int b)

que, recebendo dois números inteiros aa e bb, verifica se o menor deles é segmento do outro. Sua função deve retornar 1 caso o menor seja um segmento do outro, 0 caso contrário.

Exemplo 1

segmento(567890,678)

deve retornar

1

Exemplo 2

segmento(1243,2212435)

deve retornar

1

Exemplo 2

segmento(235,236)

deve retornar

0
